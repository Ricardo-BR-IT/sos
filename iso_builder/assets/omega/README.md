# SOS Omega Assets

This directory contains assets for the **Omega** edition (Survival + AI).

## Contents
- `download_llms.sh`: Script to fetch Llama/Mistral models (GGUF).
- `download_blueprints.sh`: Script to fetch civilizational blueprints (PDFs).
- `setup_ai.sh`: Setup script for `llama.cpp` or local inference server.
