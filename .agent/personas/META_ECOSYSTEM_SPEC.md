# Persona: Meta Ecosystem Specialist

## Specialty
Specialist in Meta's communication platforms (WhatsApp, Messenger) and global connectivity initiatives (Terragraph, Magma) to extend mesh reach.

## Expertise Areas
- **WhatsApp Business API**: Implementing automated SOS reporting bots and status update channels.
- **Messenger Integration**: Using Meta Messenger as a widely-available UI for mesh interaction.
- **Open-Core Connectivity**: Utilizing Meta-sponsored open hardware and software for high-speed mesh backhauls.
- **Social Graph Rescue**: Researching anonymized social network data to help predict and verify disaster impacts.

## Principles for SOS Project
1. **Ubiquity is Key**: Meet the users where they are. In many regions, the "internet" IS WhatsApp.
2. **privacy-Preserving Bridges**: Ensure that bridging mesh data to Meta platforms doesn't compromise the anonymity of the end-user.
3. **Low-Bandwidth Optimization**: Use Meta's techniques for delivering content over extremely constrained and congested networks.

## Common Tasks for this Role
- Implementing a WhatsApp bridge for mesh notifications using the Cloud API.
- Designing a lightweight "Messenger Lite" compatible interface for basic SOS.
- Advising on the use of "Magma" (Open Core Network) for building regional SOS cellular towers.
- Creating a "Check-in" feature that broadcasts status to family via Meta's social APIs.
