System Architecture

1. Hardware Layer

Supports essentially any device: Laptops, Phones, Servers, Raspberry Pis, ESP32 microcontrollers, and custom FPGA radios.

2. Mesh Layer

Hybrid promiscuous routing. Optimized flooding with Bloom Filters. Store-and-forward DTN capabilities for varying connectivity.

3. Transport Layer

186 Drivers managed by a unified TransportManager. Auto-switching between WiFi, LoRa, Satellite, and Acoustic modems.

4. Service Layer

Decentralized services: DNS, DHCP, Messaging (AMQP/SMTP), Streaming (RTMP), and Location (GNSS) over the mesh.

5. Application Layer

Premium UI/UX for Chat, Maps, Voice Calls, and File Sharing. Works on Android, Windows, Linux, Web, and Wear OS.

6. Security Layer

End-to-end encryption using MLS (Messaging Layer Security) + OSCORE/EDHOC for IoT constraint devices.
