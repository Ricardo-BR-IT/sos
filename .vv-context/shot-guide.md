# 分镜技巧指南

## 1. 分镜的基本构成要素

分镜是将文字剧本转化为一系列可视化镜头的基本单位，是指导拍摄的蓝图。其核心要素可通过下表快速掌握：

| 要素名称 | 描述与功能 |
|---------|-----------|
| **镜号 (Shot Number)** | 镜头的顺序编号，决定了叙事序列。编辑时需按此序列进行，但拍摄时可不必依序。 |
| **景别 (Shot Size)** | 镜头取景范围，直接影响观众对信息的接收焦点与情感距离。主要包括远景（展现场景全貌）、全景（展现人物全身及环境）、中景（人物膝盖以上）、近景（胸部以上）、特写（肩部以上或物体局部）。 |
| **镜头运动 (Camera Movement)** | 摄影机的运动方式，是营造动态视觉体验的关键。常用箭头符号表示方向。 |
| **画面内容 (Content)** | 对镜头内视觉元素的具体描述，包括场景、人物动作、表情、道具等。 |
| **声音 (Audio)** | 包含人声（对白/独白）、音乐（背景乐及起止点）和音效（环境声、动效），用于创造沉浸感。 |
| **时长 (Duration)** | 每个镜头的预计持续时间，通常以秒为单位，是控制影片节奏的重要依据。 |

## 2. 核心镜头语言技巧

### 2.1 景别的叙事功能

- **远景/全景**：常用于开场，建立场景空间感和人物与环境的关系。
- **中景**：最常用的叙事景别，平衡展示人物动作和部分环境，适用于对话和互动。
- **近景/特写**：强调人物细微表情、关键物体或重要细节，能强烈引导观众注意力，营造紧张或投入的情绪。

### 2.2 镜头运动的动态表达

#### 推镜头 (Truck In)
- 镜头向主体靠近，景别从大变小
- 用于强调主体或聚焦细节，节奏由松到紧

#### 拉镜头 (Truck Out)
- 镜头远离主体，景别由小变大
- 用于展现更多环境信息，常作为场景结束，节奏由紧到松

#### 摇镜头 (Pan)
- 机位固定，镜头水平或垂直摆动
- 用于横向展示广阔环境或跟随主体运动

#### 移镜头 (Dolly) & 跟镜头 (Follow)
- 机位本身移动
- 移镜头是摄像机在空间中移动
- 跟镜头是持续跟随一个运动的主体

#### 甩镜头 (Swish Pan)
- 快速摇摄，使画面模糊
- 常用于表现场景间的快速转换或主观视角的迅速移动

### 2.3 视角的情感暗示

- **平视 (Eye-level)**：最自然的视角，给人以平稳、客观的感觉。
- **俯视 (High Angle)**：拍摄镜头高于主体，使主体显得弱小、被动或具压迫感。
- **仰视 (Low Angle)**：拍摄镜头低于主体，使主体显得高大、强势或具权威性。

## 3. 视觉叙事与节奏技巧

### 3.1 轴线原则 (180-Degree Rule)
为保证空间连贯性，摄像机应在场景轴线的一侧180度范围内设置机位，避免使观众产生方向混淆。

### 3.2 转场技巧 (Transitions)

- **切 (Cut)**：最常用的无技巧转场，直接切换镜头，干净利落。
- **淡入/淡出 (Fade In/Out)**：画面逐渐显影或变黑，常用于段落或故事的开始与结束。
- **叠化 (Dissolve)**：前一个画面逐渐消失的同时后一个画面逐渐显现，常用于表示时间流逝或场景柔和转换。

### 3.3 节奏控制
通过镜头时长的长短变化（短镜头快切营造紧张感，长镜头维持舒缓情绪）和景别的交替运用来控制叙事节奏。

## 4. 分镜绘制的实用技巧

- **箭头标注**：是表示镜头运动（推、拉、摇、移方向）和演员走位最直观的方法。
- **多帧画面**：用于表现复杂的快速或慢速变焦效果。
- **速度线**：在甩镜头的绘制中，用于表现快速的运动感。

## 5. 专业镜头术语参考

### 5.1 景别（Shot Size）

#### 基础景别
- **大远景 (Extreme Long Shot / ELS)**: 展现宏大场景，强调环境，人物很小
- **远景 (Long Shot / LS)**: 展现完整场景，人物全身可见，强调环境关系
- **全景 (Full Shot / FS)**: 展现主体全貌，人物全身，环境清晰
- **中全景 (Medium Full Shot / MFS)**: 人物膝盖以上，展现动作和姿态
- **中景 (Medium Shot / MS)**: 人物腰部以上，突出动作和表情
- **中近景 (Medium Close-up / MCU)**: 人物胸部以上，强调表情和情绪
- **近景 (Close-up / CU)**: 人物肩部以上或物体局部，突出细节
- **特写 (Extreme Close-up / ECU)**: 聚焦关键细节（眼睛、手、物体局部），增强视觉冲击

#### 使用场景
- **远景/全景**: 开场、转场、展现环境
- **中景**: 对话、动作、展现人物关系
- **近景/特写**: 强调情绪、细节、产品展示

### 5.2 镜头运动（Camera Movement）

#### 基础运动
- **固定 (Static / Locked)**: 镜头固定不动，稳定、专注
- **推镜 (Dolly in / Push in)**: 镜头向前推进，强调主体，增强紧张感
- **拉镜 (Dolly out / Pull out)**: 镜头向后拉远，展现环境，增强空间感
- **摇镜 (Pan)**: 水平转动（左右），跟随动作或展现横向空间
- **倾斜 (Tilt)**: 垂直转动（上下），展现纵向空间或跟随垂直动作
- **升降 (Crane / Boom)**: 垂直升降，改变视角高度，增强视觉冲击

#### 高级运动
- **环绕 (Orbit / Arc)**: 围绕主体旋转，多角度展现，增强立体感
- **跟随 (Follow / Tracking)**: 跟随移动对象，保持相对位置，增强动感
- **变焦推拉 (Zoom in/out)**: 改变焦距，快速聚焦或拉远（注意与推拉镜的区别）
- **手持 (Handheld)**: 手持拍摄，营造真实感、紧张感
- **滑轨 (Slider)**: 横向或纵向滑动，平滑过渡

### 5.3 镜头角度（Camera Angle）

#### 水平角度
- **平视 (Eye Level)**: 正常视角，自然真实，常用角度
- **正面 (Front)**: 正对主体，直接、正式
- **侧面 (Profile)**: 侧面拍摄，展现轮廓
- **背面 (Back)**: 背面拍摄，营造神秘感

#### 垂直角度
- **俯拍 (High Angle)**: 从上往下，显得主体渺小或展现全貌
- **仰拍 (Low Angle)**: 从下往上，显得主体高大或增强气势
- **鸟瞰 (Aerial / Top-down)**: 垂直向下，展现布局和空间关系
- **倾斜角度 (Dutch Angle / Canted)**: 倾斜画面，营造紧张、不稳定感

### 5.4 音效分类（Sound Effects）

#### 环境音效（Ambient）
- **自然音**: 风声、雨声、雷声、海浪声、鸟鸣
- **城市音**: 车辆行驶声、人群喧哗声、建筑工地声
- **室内音**: 空调声、钟表声、房间回声

#### 动作音效（Foley）
- **脚步声**: 走路、跑步、跳跃
- **物品声**: 门开关、物品碰撞、纸张翻动
- **身体声**: 呼吸、心跳、衣服摩擦

#### 特殊音效（SFX）
- **爆炸声**: 爆炸、撞击、破裂
- **科技音**: 电子设备、机械运转、科幻效果
- **魔法音**: 魔法效果、超自然现象

### 5.5 音乐术语（Music）

#### 音乐类型
- **背景音乐 (BGM)**: 轻快、舒缓、紧张、激昂
- **主题音乐 (Theme)**: 与主题匹配的旋律
- **环境音乐 (Ambient)**: 营造氛围，不突出

#### 音乐参数
- **BPM (Beats Per Minute)**: 节奏快慢
  - 慢速: 60-80 BPM（舒缓、温馨）
  - 中速: 80-120 BPM（正常、活跃）
  - 快速: 120+ BPM（紧张、激昂）
- **音量控制**: 百分比表示（20%-50%常见）
- **处理方式**: 渐入(Fade in)、渐出(Fade out)、突然切入(Cut in)

## 总结

分镜的本质是视觉化的叙事语法。AI在处理分镜任务时，应以上述要素和技巧为框架，致力于将抽象的文本信息转化为准确、连贯且富有表现力的视觉序列，确保每个镜头都服务于整体故事的讲述。
